# react-widget

## How to build react-widget

```sh
npm i
npm run build
```

## How to run react-widget storybook

```sh
npm i
npm run storybook
```

## How to run react-widget example

```sh
npm i
npm run build
cd example/
npm i
npm run start
```

## How to publish react-widget to npm

```sh
npm i
npm run build
npm login
npm publish
```
